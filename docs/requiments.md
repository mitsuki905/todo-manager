# ToDoアプリ 要件定義書

## 1. 概要

* アプリ種別：Windowsデスクトップアプリケーション
* 開発言語：Python
* GUIライブラリ：tkinter
* 利用者：個人利用

***

## 2. データ保存

* 保存形式：JSONファイル
* ファイル名：`tasks.json`
* 永続化：ローカルファイルに保存

***

## 3. データ構造

### タスク項目

| 項目名         | 型      | 必須 | 説明        |
| ----------- | ------ | -- | --------- |
| id          | int    | ○  | 一意識別子     |
| title       | string | ○  | タイトル      |
| description | string | ×  | 詳細説明      |
| priority    | string | ○  | 高 / 中 / 低 |
| status      | string | ○  | 未完了 / 完了  |
| created\_at | string | ○  | 作成日時      |
| updated\_at | string | ○  | 更新日時      |

***

## 4. 機能要件

### 4.1 タスク管理

* タスク追加
* タスク編集
* タスク削除
* タスク一覧表示
* タスク完了切替（未完了 ⇔ 完了）

### 4.2 操作仕様

* タスク選択：クリック
* タスク編集：ダブルクリック
* 完了操作：ボタンまたはチェック操作

***

## 5. UI要件

### 5.1 画面構成

* 単一画面構成

### 5.2 表示形式

* 表形式（Treeview）で表示
* 表示項目：
  * タイトル
  * 優先度
  * 作成日
  * 状態

### 5.3 UIコンポーネント

* タスク一覧（テーブル）
* 操作ボタン
  * 追加
  * 削除
  * 完了

***

## 6. 表示仕様

### 6.1 並び順

* 作成順

### 6.2 優先度表示

* 高：赤色
* 中：オレンジ色
* 低：緑色

***

## 7. 状態管理

* 初期状態：未完了
* 操作により「完了」に変更可能
* 再操作で未完了に戻す

***

## 8. 入力仕様

### 8.1 必須入力

* タイトル

### 8.2 任意入力

* 詳細説明
* 優先度（未指定時は「中」）

***

## 9. 非機能要件

* ローカル環境で動作
* オフライン利用可能
* 軽量かつシンプルなUI

***

## 10. 拡張性（考慮事項）

* 項目追加（例：締切）の拡張可能
* 並び替え機能の追加可能
* UI改善の余地あり

# ToDoマネージャー

シンプルで使いやすい Windows 向けデスクトップ ToDo 管理アプリです。  
タスクの追加・編集・削除・完了管理をかんたんに行えます。

---

## 画面イメージ

| 項目 | 内容 |
|------|------|
| タイトル | タスクの名前 |
| 優先度 | 高 / 中 / 低（色分け表示） |
| 作成日 | タスクを登録した日付 |
| 状態 | 未完了 / 完了 |

---

## 動作環境

- **OS**：Windows 10 / 11
- **Python**：3.10 以上
- 追加ライブラリのインストールは不要です（Python 標準ライブラリのみ使用）

---

## セットアップ

初回のみ、以下の手順でセットアップを行ってください。

1. `run_setup.bat` をダブルクリックして実行します。
2. 「セットアップ完了」と表示されれば準備完了です。

> Python がインストールされていない場合は、[Python 公式サイト](https://www.python.org/downloads/) からインストールしてください。

---

## 起動方法

`run_app.bat` をダブルクリックするとアプリが起動します。

---

## 使い方

### タスクを追加する

1. ツールバーの **「＋ 追加」** ボタンをクリックします。
2. ダイアログが開くので、以下の項目を入力します。
   - **タイトル**（必須）：タスクの名前を入力します。
   - **詳細説明**（任意）：タスクの詳しい内容を入力できます。
   - **優先度**（任意）：「高」「中」「低」から選択します。未選択時は「中」になります。
3. **「OK」** ボタンをクリックすると、タスクが一覧に追加されます。

### タスクを編集する

1. 一覧からタスクを **ダブルクリック** します。
2. 編集ダイアログが開くので、内容を変更します。
3. **「OK」** ボタンをクリックすると変更が保存されます。

### タスクを完了にする / 未完了に戻す

1. 一覧からタスクをクリックして選択します。
2. ツールバーの **「✔ 完了切替」** ボタンをクリックします。
3. 完了済みのタスクはグレー表示になります。再度クリックすると未完了に戻ります。

### タスクを削除する

1. 一覧からタスクをクリックして選択します。
2. ツールバーの **「🗑 削除」** ボタンをクリックします。
3. 確認ダイアログで **「はい」** を選択すると削除されます。

---

## 優先度の色分け

| 優先度 | 表示色 |
|--------|--------|
| 高 | 赤色 |
| 中 | オレンジ色 |
| 低 | 緑色 |
| 完了済み | グレー |

---

## データの保存場所

タスクのデータはアプリと同じフォルダ内の `tasks.json` に自動保存されます。  
アプリを閉じてもデータは保持されます。
